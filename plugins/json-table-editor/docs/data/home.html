JSON Editor is a collection of jquery functions to convert a json array to an editable html table and then back to jason array

json array should be in a below format

[{'key1':'value1_A', 'key2':'value2_A'},{'key1':'value1_B', 'key2':'value2_B' },{'key1':'value1_C', 'key2':'value2_C' }]

This is an array of javascript object and each object is a list of key & value pairs

Some features of the code

  • In a key value pair; the value can be a single string/number or can be an object or array of objects as well. In this case the created table will have a child table in place of value.
  • In the editor you can insert new values, columns, rows, tables and delete the existing ones
  • The output JSON is simplified in such a way that if a key doest not contain a value, it is removed from the resultant JSON
  • The editor interface is created to be very user friendly, one can just right click and performs the operations

How to use

After including the javascript file in you code just call the below function

jsonEditorInit(table_container_id, json_input_container_id, json_output_container_id, json_to_table_btn_id, table_to_json_btn_id);

  • table_container_id: id of the container where the editor will be created
  • json_input_container_id: id of the input element from where the json array is to be taken
  • json_output_container_id: id of the output container where the output json is to be shown
  • json_to_table_btn_id: id of the button which will be used to initiate the json to table call.
  • table_to_json_btn_id: id of the button which will be used to initiate the table to json call.
